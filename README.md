## 🌍 Overview

Weather Forecast is a lightweight front-end web application that fetches live weather data and displays it in a clean, intuitive interface. No frameworks, no build tools — just pure HTML, CSS, and JavaScript.

## 🚀 Getting Started

### Prerequisites

Just a modern web browser — no installs needed!

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/itxashancode/Weather-Forecast.git

# 2. Navigate into the folder
cd Weather-Forecast

# 3. Open in your browser
open index.html
```

### 🔑 API Key Setup

This app uses the [OpenWeatherMap API](https://openweathermap.org/api).

1. Sign up for a free key at [openweathermap.org](https://openweathermap.org/)
2. In `index.html`, find and replace:

```js
const apiKey = "YOUR_API_KEY_HERE";
```
